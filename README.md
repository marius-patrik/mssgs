# mssgs

> Beeper-like unified messaging inside VSCode.

## Features

- Unified inbox for iMessage, Telegram, WhatsApp, and Instagram.
- Add accounts through a simple setup wizard.
- Beautiful React-based UI with motion and theming.
- Local SQLite cache for offline access and fast search.
- Public-release safe: uses official Matrix bridges, no scraping.

## Development

```bash
npm install
npm run build
npm run test
npm run package
```
