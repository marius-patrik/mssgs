# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-27

### Added

- Initial release of the mssgs VSCode extension.
- Activity-bar messenger webview built with React and Rsbuild.
- Unified conversation inbox with search, pinning, favorites, and archiving.
- Thread view with message bubbles, reactions, read receipts, typing indicators, attachments, and composer.
- Account setup wizard for adding chat accounts.
- Matrix backend connection with a typed message bus.
- Service detection and registry for cross-service conversations.
- SQLite cache for accounts, conversations, and messages.
- Keyboard shortcuts and command-palette commands for common actions.
- Reminders and scheduled messages.
- Vitest unit tests, VSCode integration tests, and Playwright smoke tests.
- CI/CD workflows for lint, typecheck, test, build, package, and release.
